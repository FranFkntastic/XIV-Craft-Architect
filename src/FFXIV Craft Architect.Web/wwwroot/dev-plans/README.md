Drop `.craftplan` files here while running the Blazor app in Development.

Use `Tools > Load Dev Craftplan...` and enter either the filename, for example:

```text
shark-class-bridge-16.craftplan
```

or a path relative to `wwwroot`, for example:

```text
dev-plans/shark-class-bridge-16.craftplan
```

Files in this folder are local development fixtures and should not be treated as user saves.
